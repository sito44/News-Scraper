# News-Scraper

![Image of Mongo](https://i2.wp.com/www.damiencosset.com/wp-content/uploads/2017/07/mongodb.png?w=413&ssl=1)

## Description

Web scraper app that uses Mongoose and Cheerio to collect news articles off a given website.

## Technologies
- Javascript
- Node.js
- Express.js
- MongoDB
- Mongoose
- Cheerio
- Handlebars
